Учебная консольная утилита для изучения побитовых операций и базовой обфускации строк (XOR).


## Что делает

Принимает текстовую строку, шифрует каждый символ по числовому ключу через XOR, выводит результат в виде шестнадцатеричного массива.


## Зачем

Лабораторная работа для практического понимания побитовых операций.


## Дисклеймер
Учебный проект, не предназначен для использования в реальных вредоносных сценариях.
